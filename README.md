  - Dark Mode
  - Modern UI
    - Clean Colours
  - Custom Themes
    - Colours
    - Backgrounds

The latest version of SEQTA Vanced can be installed from the Chrome Web Store. **[Click here to view.](https://chrome.google.com/webstore/detail/seqta-vanced/kekoobmbfdkempioiihnepdnapdpdaok)**
